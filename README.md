# screentool.lua
A Linux tool to take screenshots and screencaps under X11. Depends on ximgcopy, ffmpeg, slop, maim, xrandr and possibly a bunch of other stuff. todo: fix this thing.
## Usage
`screentool.lua <action> [...action]`
### Actions
* `measure`
* `screenshot`
* `save`
* `copy`
* `upload`
* `screencap`
* `stop`
* `compress`
